Weather to Wear

This project was created in React with a Rails API. This app uses the temperature and chance of precipitation to recommend a jacket or umbrella to users. Logged in users are able to save cities to their settings, as well as adjust their sensitivity to cold and inclination to carry an umbrella.

Recipe Manager

This project was created using jQuery and Rails. Recipe's feature key-ingredients, which users can use to search for other recipes using the same ingredient(s), as a means of being resourceful.

Frogger Clone

The Frogger clone game is created in JavaScript. Users can use the ←, ↑, →, ↓ keys on keyboard to control the character and move across to the water.

Neighborhood Map

Neighborhood map app is created in React. This app features my favorite eateries at Universal Studios Orlando by marking each eatery using Google Maps JavaScript API and FourSquare API.

Restaurant Reviews

Restaurant reviews app is created in JavaScript. This project showcases responsive design, a11y using ARIA roles and labels, as well as offline browsing through service worker. Map API provided by Mapbox.

My Bookshelf
